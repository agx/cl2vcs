cl2vcs parses and displays changelogs of Debian packages. If the package's changelog lists Vcs commit ids and if the package's debian/control file has a Vcs-Browser field, cl2vcs adds links from the changelog back to the version control system for easy patch review. A real live package that supports this is libvirt.
